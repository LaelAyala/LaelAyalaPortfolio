Fabricating a Stacking robot

I used SolidWorks and machining techniques to build a robot that could stack a cube, a ring, and a sphere on top of each other

For this class, we had a year long project in which we built a robot from scratch. We had to machine the parts and assemble the gearboxes ourselves in order to create a working robot. The robot would then compete against other robots for who could put the most objects(rings, spheres, and cubes) on various levels of a shelf. However, you got bonus points for stacking the objects, so my team designed a robot that used a trap door technique that scooped the objects off the turf and was controlled enough to be able to stack the objects on top of each other.

Through this project, I learned how to design and assemble parts using SolidWorks, 3D Print, lasercut, use a CNC Mill, band saw, and a plethorea other tools
